Advanced Task: Targeted Profiling
To increase success rates, I researched CUPP (Common User Passwords Profiler). Unlike a simple dictionary attack, a targeted attack uses "Seed Words" from the victim's life.

Example Logic:
If Name = "Nur" and DOB = "2001", the attacker generates:
[Name][Year], [Name][SpecialChar][Year], [Initial][Year].
This significantly reduces the time needed compared to a full brute-force of all character combinations.
